# [PlantUML](https://plantuml.com/)  software diagraming tool

## Pre-requisites
1- Java 8 or higher  

## Installation
1- Graphviz:       
 -- MacOS: `$ brew install graphviz`  
 -- Ubuntu: `$ sudo apt install graphviz`  
2. Go to `VSCode Settings > Extensions` - search for and install `PlantUML` by `jebbs.plantuml`.

## Usage
Preview diagram:            
- `Option + D` (MacOS)   
- `Alt + D` (Ubuntu)
